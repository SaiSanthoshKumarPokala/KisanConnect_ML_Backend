How it works

  • Capture mic audio with MediaRecorder.
  • POST the audio file to /chat.
  • Groq ASR transcribes the speech.
  • FAISS retrieves the top 2 context chunks.
  • Groq LLM generates a concise same-language answer.
  • Sarvam TTS returns base64 audio for playback.
